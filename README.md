# SpanAISolutions.com

Premium one-page Astro marketing site for Span AI Solutions.

## Stack

- Astro
- Vanilla CSS
- Playwright smoke tests
- Netlify deployment via GitHub

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run check
npm run build
npm run test:e2e
```

## Deployment

Pushes to `main` trigger Netlify production deploys for:

- `https://spanaisolutions.netlify.app`
